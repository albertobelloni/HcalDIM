Detailed Description

Error checking class for FED data.

This class checks FED data for errors. A set of error counters is maintained as a set of public members, which may be accessed for histogram filling.

Various levels of checking are suppored, requiring differing amounts of CPU time.

Check FED data for errors.

Parameters
fedUnpacked FED object to check
toCheckwhat to check (any from list below with |) Default is (check_DCC|check_HTR)
  • check_None = don't check anything (initialize empty structure)
  • check_DCC = check DCC/CDF errors
  • check_HTR = check HTR header errors
  • check_CRC = calculate and check CRC
  • check_Digi = check all errors including capIDs and FEE links
