Detailed Description

Manage data integrity plots for HCAL.

Setup and fill a set of histograms and plots for HCAL data integrity checking. Expects that actual ROOT TH and TGraph objects are created elsewhere, i.e. in a CMSSW job or stand-alone program. This class handles setting the labels, bins etc and filling in the data. The rendering of the plots or storing in a root file is again handled outside this class.

Parameters
nFedsnumber of FEDs to plot (0 for no by-FED plots)
Fedslist of FED numbers (700-731 inclusive)
overViewPlot2D histogram for overview plot (NULL to omit plot)
fedHtrPlot2D histogram for error plot by FED/HTR
TTSstatePlotsplots of TTS state versus event number
HTRDigiPlotsplots of HTR QIE data state versus channel
dataSizePlotsplots of data size versus event number
trigRatePlotsplots of trigger rate versus event number
dataRatePlotsplots of data rate versus event number
debugenable debug output

The pointers should each point to an array of size nFeds, of pointers to already-created ROOT objects of the specified type. If nFEDs=0, NULL pointers may be supplied. The number of bins or points need not be set... they will be set in by this method.
