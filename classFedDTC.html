Detailed Description

Class to access items within HCAL FED Data with minimal unpacking.

The constructor takes a pointer to the raw data and it's length. An array of pointers to UHTR payloads is set up for quick access. Accessor functions are provided to the various parts of the data payload. An array of pointers to each UHTR's channel data is calculated on demand.

Some error-checking functions are provided.

NOTE! Does not currently take into account format variations due to UHTR firmware revisions.
