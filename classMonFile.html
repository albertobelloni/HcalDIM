Detailed Description

Class for accessing HCAL MonLogger text files.

Author
Eric Hazen (Boston University)

This class provides access to MonLogger text files written by the HCAL online software. It makes a few assumptions about the file format: There must be a "timestamp" column. The timestamps must be close to time order.
